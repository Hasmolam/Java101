# MinMaxNumberFinder

This Java program finds the minimum and maximum numbers from a series of numbers entered by the user.

## Usage

1. Compile the program: `javac MinMaxNumberFinder.java`
2. Run the program: `java MinMaxNumberFinder`
3. Enter the number of numbers you will enter when prompted.
4. Enter each number when prompted.
5. The program will calculate and display the minimum and maximum numbers entered.

## Notes

- The program uses a `Scanner` object to read user input for the number of numbers and each number.
- It iteratively compares each number to find the minimum and maximum using a simple comparison approach.
- It assumes all numbers entered are integers.
